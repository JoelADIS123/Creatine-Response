How to setup and run the project :

This project predicts whether an individual is a Responder (1) or Non-Responder (0) to creatine supplementation using physiological and lifestyle parameters.
It uses a Random Forest Classifier built with Python and scikit-learn.

Software:
Python 3.8 or above
VS Code
Libraries:

Install required packages using:
pip install pandas numpy matplotlib scikit-learn

Setup Instructions
Step 1: Create Project Folder
mkdir creatine-project
cd creatine-project
Step 2: Add Files
Place creatine_data.csv in the folder
Create main.py and paste the provided code
Step 3: Verify Files
Ensure structure:
creatine_data.csv
main.py

Running the Project:
Run the script using:
python main.py
